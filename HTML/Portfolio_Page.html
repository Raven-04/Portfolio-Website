Plant Disease Detection using Pooling-based Vision Transformers (PiT)

  • Developed as a final project for the Computer Vision course, this project explores the application of the Pooling-based Vision Transformer (PiT) architecture for identifying plant diseases.

  • Using the comprehensive Plant Village dataset of 70,000 images across 29 classes, the study benchmarks PiT against Vision Transformers (ViT) and ResNet models, evaluating accuracy, precision, recall, computational efficiency, and robustness.

  • The project's approach leverages PiT's ability to capture local and global image features, aiming to provide a scalable and efficient solution for automated agricultural diagnostics.
